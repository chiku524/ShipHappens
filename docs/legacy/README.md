# Legacy design (v0.2)

This folder contains the **superseded** ShipHappens design from the Godot era: a 6–8 player **Crew vs Stowaway** social-deduction freight co-op.

The project pivoted in 2026 to **Vault Break** — competitive co-op escape room tournaments. See the current design:

## What carries forward

- **Cartoon space-freight tone** and HR comedy
- **3D assets** in `assets/models/` (crane, breakers, coolant, shuttle bay, etc.)
- **Power Hour breaker sequence** — reused in Breaker Panic room (extended)
- **Character roster** — see [../CHARACTERS.md](../CHARACTERS.md)

## What does not carry forward

- Stowaway role assignment and hidden traitor loop
- 20-minute single-round session structure
- Emergency meetings and Write-Up votes as core loop
- "Complete 7 of 10 jobs to launch shuttle" win condition

Some ideas may return as optional room modifiers (e.g. Glitch Socket saboteur).
