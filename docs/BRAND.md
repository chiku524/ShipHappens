# Brand — Pugdy Monsters

## Locked names

| Layer | Name |
|-------|------|
| Universe / IP | **Pugdy Monsters** |
| Game (full) | **PugdyMon: Party Saga** |
| Short | **PugdyMon** |
| Full mini-game circuit | **Party Saga** (Race → Vibe → Shooter) |
| Social hub | **The Nest** |

Spelling is **Pugdy** (not “Pudgy”) — intentional brand mark for this title.

## Tone

Cute, chunky creatures in a party playground. Family-friendly toy weapons, collectible skins on Boing Network, season “Hype” as Nest Points.

## Do not

- Lead with the old ShipHappens / vault / freight framing in player-facing UI.
- Imply real-money gambling; Boing rewards are collectibles / soft currency.
